# yaffs2-recovery #

***

### Discription ###

Implementation of historical data recovery on YAFFS2 filesystem.

***

### Installation

```sh
make
```

***

### Usage ###

```sh
yaffs2-recovery [-f <file>] [-d <dest>] [-h]
  -f <file>   Recover from img <file>
  -d <dest>   Write recovered files to <dest> directory
  -h          Print help
```
